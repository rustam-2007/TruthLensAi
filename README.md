## 📌 Project Overview

**TruthLens AI** is a professional-grade, full-stack visual forensics suite built to analyze, reveal, and audit digital assets for generative AI modifications, structural splices, and manipulation anomalies. Utilizing a clean, high-contrast cyber-forensics dark grid layout, it provides real-time client-side graphics calculations paired with deep server-side metadata and AI-driven validation.

---

## ✨ Core Forensic Features

### 🖥️ Real-time Graphic Amplifiers (Client-Side)
*   **Error Level Analysis (ELA):** Amplifies compression ratio variances to highlight non-homogeneous boundaries (indicates spliced components or generative infill).
*   **Chroma Channel Isolation:** Isolates specific high-frequency color bands to identify localized digital airbrushing or brush touch-ups.
*   **Contour Edge Diagnostics:** Scans high-frequency pixel edge gradients to trace structural integrity, shadows, and perspective mismatches.

### 🎛️ Sandbox Comparison Modes
*   **Active Comparison Slider:** Drag-and-wipe layout to instantly compare original source pixels with localized forensic analysis maps.
*   **Dual-Panel Parallel Workspace:** Side-by-side active analysis for deeper comparative analytics.
*   **Dynamic Target Markers:** Visual interactive markers flagging facial asymmetry, pupil reflection failures, noise inconsistencies, and structural anomalies.

### 🛡️ Production-Grade Network Resilience
*   **Graceful Degradation:** Automatic diagnostics for missing environment keys (such as `GEMINI_API_KEY`) or upstream `503 Service Unavailable` API congestion.
*   **Interactive Safeguard Backups:** Falls back immediately to high-fidelity localized heuristic models ensuring non-blocking user operations with transparent diagnostic logs.

---

## 🛠️ Technical Stack

*   **Frontend:** React 18, Vite (No-HMR static rendering), `@motion/react` for micro-animations, Lucide React (vector assets).
*   **Backend:** Node.js, Express.js server proxying critical requests to avoid API client-side leakage.
*   **Styling:** Custom configured Tailwind CSS optimized for dark high-contrast cyber-aesthetics.
